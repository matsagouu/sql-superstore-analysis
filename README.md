# SQL Superstore Analysis (SQLite)

SQL portfolio project με δεδομένα Superstore.

## Τι δείχνει
- KPI υπολογισμούς (Sales, Profit, Orders, Margin)
- GROUP BY / HAVING / ORDER BY
- Time series (πωλήσεις ανά μήνα)
- Shipping performance (avg shipping days)
- CTE για discount buckets (Discount → Profit impact)
- Window functions (ranking μέσα σε κατηγορία)

## Tech
- SQLite (online)
- SQL (aggregations, CTEs, window functions)

## Files
- `queries.sql` : όλα τα queries του project
